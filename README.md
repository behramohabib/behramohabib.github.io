# Personal Site

Minimal static writing site with two shelves: personal and professional.

## Structure

- `index.html` landing page
- `personal/` personal writing
- `professional/` professional writing
- `styles.css` shared styles

## Add writing

Create a new HTML file in `personal/` or `professional/` and link it from the
folder's `index.html` list. Keep the layout or customize as needed.
